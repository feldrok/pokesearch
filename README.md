
# Pokesearch

Just a small project to test skills for a challenge task.


## Run Locally

Clone the project

```bash
  git clone https://github.com/feldrok/pokesearch.git
```

Go to the project directory

```bash
  cd pokesearch
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
